How This Was Built

This site is maintained by a WhatsApp bot that manages the Sampson-Kahn family tree. The idea started after I read a blog post about building a family tree AI assistant that connected a GEDCOM file to a WhatsApp bot using a graph database. I thought: I could do something like that for our family.

The bot is built with Baileys for WhatsApp connectivity and the Anthropic SDK for Claude as the language model. The family data lives in a structured JSON file with fields for names, dates, places, relationships, and sources.

It connects to Geni.com through their API to look things up, validate data, and push updates when family members share new information. It can also export the tree as a GEDCOM file for use in other genealogy software.

The bot listens to our WhatsApp family group, answers questions about family history, and updates the tree interactively. Changes get saved to the JSON, synced to Geni, exported to GEDCOM, and published to this site. The narrative on the "Our Story" page is generated using Claude, and the whole site deploys automatically through GitHub Pages.
